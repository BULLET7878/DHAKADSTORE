# Ticket Booking App

A full-stack **ticket booking application** built using **React** for frontend and **Express + SQLite3 + Prisma** for backend. The backend is deployed on **Render**, and the frontend is deployed on **Vercel**.  

This app is **flexible and can handle any type of ticket**: movies, events, concerts, sports, travel, or any custom ticket system.  

---

## **Tech Stack**

- **Frontend:** React, Vite (or Create React App)  
- **Backend:** Node.js, Express  
- **Database:** SQLite3  
- **ORM:** Prisma  
- **Deployment:** Backend on Render, Frontend on Vercel  
- **Other:** CORS handled via `cors` middleware  

---

## **Features**

- **Create Tickets:** Book tickets by providing details.  
- **Read Tickets:** View all booked tickets or a specific ticket.  
- **Update Tickets:** Edit ticket information.  
- **Delete Tickets:** Cancel a booked ticket.  
- **RESTful API:** Fully implemented CRUD endpoints.  
- **Flexible Ticket Types:** Works for movies, concerts, sports events, travel, or any custom ticket system.
